# Basic to Intermediate C Practice 💻

This repository contains my personal journey of learning C programming, starting from fundamental concepts to intermediate-level problem solving and projects.

## 📚 Topics Covered
*   **Basics:** Syntax, Variables, Data Types, and Operators.
*   **Control Flow:** If-else statements, Switch cases, and Loops (for, while, do-while).
*   **Functions:** Function declarations, definitions, and recursion.
*   **Arrays & Strings:** Handling 1D/2D arrays and string manipulations.
*   **Pointers:** Understanding memory addresses, pointer arithmetic, and DMA.
*   **Structures & Unions:** Custom data types and memory management.
*   **File Handling:** Reading from and writing to files.

## 🚀 Purpose
The goal of this repository is to document my progress, store solved problems from various platforms, and build a strong foundation in low-level programming.

## 🛠️ How to Compile and Run
To run any of the programs, you can use a C compiler like **GCC**:
1. Open your terminal/command prompt.
2. Compile the file: `gcc filename.c -o filename`
3. Run the executable: `./filename` (Linux/Mac) or `filename.exe` (Windows)
